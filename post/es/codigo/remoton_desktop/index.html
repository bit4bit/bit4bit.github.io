Remoton - Desktop

Usa Remoton y XPRA para crear una plataform de acceso remoto, Un reemplazo libre para TeamViewer/Logmein/etc..

Mas Detalles

Uso

Descargar

El procedimiento seria:

  • instalar remoton-server en un servidor local o publico.
  • crear certificados con remoton-server-cert
  • correr remoton-server con los respectivos certificados
  • instalar remoton-client-desktop a los usuarios que se da soporte
  • instalar remoton-support-desktop a los usuarios que dan el soporte
  • compartir el certificado cert.pem de forma segura a todos los usuarios.
