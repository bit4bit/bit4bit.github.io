glivo

GLIVO

Sencillo kit de herramientas para el desarrollo aplicaciones IVR desde el lenguaje GO y Freeswitch haciendo uso del mod’ulo mod_event_socket.

TUTORIAL

Para esto necesitamos instalar el paquete glivo.

$ go get github.com/bit4bit/glivo

Inicializaci’on de Glivo

El procedimiento general Glivo es crear un socket en la interfaz y puerto al que se comunicar’a el Freeswitch, para esto usamos la funci’on glivo.NewSession del paquete en conjunto con net.Listener o bien podemos directamente la funci’on glivo.Listen del paquete Glivo, miremos:

glivo.NewSession

package main

import (
	"net"
	"log"
	"github.com/bit4bit/glivo"
)

func main() {
	listen, listenErr := net.Listen("tcp", "localhost:5050")
	if listenErr != nil {
		log.Fatal(listenErr)
	}
	
	socket = glivo.NewSession(listen)
	defer socket.Close()
}

glivo.Listen

Nos permite abrir el la direcci’on TCP indicada directamente.

package main

import (
	"net"
	"log"
	"github.com/bit4bit/glivo"
)

func main() {
	socket, socketErr := glivo.Listen("localhost:5050")
	if socketErr != nil {
		log.Fatal(socketErr)
	}
	defer socket.Close()
}
